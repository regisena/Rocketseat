# \<input type="button">

- Representa um botão
- Usado para enviar formulários
- É estilizado pelo user agent

## Atributos
- type
    - submit
    - reset
    - button
- autofocus
- disabled
- name
- value
- form

```html
<input class="favorite styled"
       type="button"
       value="Add to favorites">

<button disabled>Enviar</button>
```
