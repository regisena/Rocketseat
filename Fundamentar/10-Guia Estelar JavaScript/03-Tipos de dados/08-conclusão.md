# Tipos de dados

- Conforme o ECMAScript (órgão que padroniza o javascript) standard temos 9 tipos de dados:

* Data types
 * Primitive / Primitive value
 * Structural
 * Structural Primitive

## Primitivos

* String
* Number
* Boolean
* undefined
* Symbol
* BigInt

## Estruturais

* Object
    * Array
    * Map
    * Set
    * Date
    * ...
* Function

## Primitivo Estrutural / Structural Root Primitive

* null
