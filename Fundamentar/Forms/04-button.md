# Button

- Representa um botão
- Usado para enviar formulários
- É estilizado pelo user agent

## Atributos
- type
    - submit
    - reset
    - button
- autofocus
- disabled
- name
- value
- form

```html
<button type="submit">Enviar</button>

<form>
    <input type="text" value="">
    <button type="reset">Limpar</button>
</form>

<button disabled>Enviar</button>
```
